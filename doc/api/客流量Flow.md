# API - 客流量

## 获取客流量
发送该请求获取指定店铺的客流量数据。

POST请求 /api/flow/shop
```yaml
shop: long 店铺id
start: date 数据开始日期 格式：yyyy-MM-dd HH:mm:ss, 例：2023-03-09 00:00:00
lasting: int 持续天数
```
